# Aura — Keyboard Shortcuts

## 1. Global Shortcuts

Available everywhere in the app.

| Key | Action |
|---|---|
| `Space` | Start / Pause / Resume focus timer |
| `Esc` | Close modal / dismiss toast / stop timer |
| `?` or `/` | Show keyboard shortcut cheatsheet modal |
| `[` | Toggle sidebar |
| `D` | Go to Dashboard |
| `F` | Go to Focus page |
| `J` | Go to Journeys |
| `H` | Go to Habits |
| `P` | Go to Planner |
| `A` | Go to Analytics |
| `S` | Go to Settings |
| `N` | New task (on Dashboard or Planner) |
| `B` | New habit (on Habits page) |

## 2. Focus Page Shortcuts

| Key | Action |
|---|---|
| `Space` | Start / Pause / Resume |
| `Esc` | Stop session |
| `1` | Switch to Pomodoro mode |
| `2` | Switch to Free mode |
| `M` | Mute / Unmute timer sounds |

## 3. Planner Shortcuts

| Key | Action |
|---|---|
| `N` | New task for selected date |
| `↑` `↓` | Navigate task list |
| `Enter` | Open selected task |
| `Delete` / `Backspace` | Delete selected task |
| `Tab` | Next field in task form |

## 4. Habits Shortcuts

| Key | Action |
|---|---|
| `B` | New habit |
| `Space` | Check/uncheck selected habit |
| `↑` `↓` | Navigate habit list |

## 5. Analytics Shortcuts

| Key | Action |
|---|---|
| `E` | Export data (CSV) |
| `R` | Refresh data |
| `1` `2` `3` | Switch chart view (Heatmap / Timeline / Treemap) |

## 6. Cheatsheet Modal

Triggered with `?` or `/`. Shows a glass modal with all shortcuts grouped by section. Dismiss with `Esc`.
